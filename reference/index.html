FAQ

常見問題

賽事日即時數據每 30 秒更新一次（賠率變動為實時推送）。賽後結果通常於賽事結束後 5–15 分鐘內更新。歷史數據則於每季結束後進行年度完整性審核。

Standard 方案每日 10,000 calls；Professional 每日 100,000 calls；Enterprise 可按需求自訂。WebSocket 連線不限次數。所有方案均可根據用量協商調整。

可以。我哋提供 14 日免費 Sandbox 環境，包含最近 3 季樣本數據同基本 API 存取。試用期間無需信用卡，適合評估數據質素同 API 整合難度。

所有方案預設提供 JSON 格式（REST API）。Professional 及以上方案額外支援 CSV 批量匯出。Enterprise 方案可選 Parquet、Avro 或直接 SQL 資料庫存取。

Standard 同 Professional 方案支援月費或年費（年費享 15% 折扣）。Enterprise 方案為年度合約，按需求定制條款。

client = horsorion.Client(api_key="hsr_live_...")

    print(f"{race.id}: {race.name} — {race.distance}m")

About

關於 Horsorion

扎根香港，專注賽馬數據。為專業投資者提供最可靠、最全面嘅賽馬資訊平台。

四十載數據 · 一個平台

Horsorion 始於一個簡單信念：賽馬數據應該被認真對待。我哋由 1980 年代起搜集、整理同驗證香港賽馬數據，至今已累積超過四十年的完整歷史紀錄。

我哋嘅團隊結合賽馬專業知識同數據工程技術，為專業 syndicate、量化基金同分析初創提供精確、即時、結構化嘅數據服務。
